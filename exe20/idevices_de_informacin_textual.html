iDevices de información textual

Para presentar contenidos o temas (incluidos imágenes, vídeos, contenidos multimedia...) tendremos los siguientes iDevices:

  • Texto libre: similar a como trabajamos con un editor de texto normal. Una vez publicado, se presenta al estudiante como la lectura de un texto. 
  • Objetivos: para incluir los objetivos  en una caja de texto destacado y con un icono que enfatiza el contenido mostrado.
  • Conocimiento previo: destinado a incluir los contenidos que el estudiante debe conocer para saber a cabo una actividad. También aparece como texto destacado y con un icono representativo.
  • Nota: destinado a incluir comentarios y anotaciones del autor. Este iDevice NO será visible en ninguno de los exportados.

¿Sabías qué?

TinyMCE es el editor de texto WYSIWYG     para HTML de código abierto utilizado en eXeLearning.

 

A continuación te mostraremos las capacidades de edición que presenta:

Primera fila

 En la primera fila encontraremos las herramientas necesarias para la edición de los textos.

Para saber más, muestra retroalimentación:

Retroalimentación

Podremos seleccionar palabras con el ratón y luego aplicarles el formato haciendo clic en el icono del editor o por el contrario, seleccionar el formato en el editor y escribir (se nos mantendrá el formato seleccionado).

Opciones:

Nuevo documento: nos permite dejar en blanco el documento, previa confirmación.

Corrector ortográfico: nos facilita detectar las palabras escritas de manera incorrecta. Para su correcto funcionamiento, deberemos tener instalado el corrector ortográfico/diccionario del navegador.

Negrita (Ctrl+B), cursiva (Ctrl+I) y subrayado (Ctrl+U). 

Tachado: nos permite añadir una línea horizontal que atraviesa por su centro la palabra o palabras seleccionadas.

Alineación: nos permite la alineación de un párrafo a la izquierda, centrado, a la derecha o justificado.

Listas: nos facilita la creación de listas con viñetas (círculo, disco y cuadrado) y ordenadas (alfabeto y numérico).

 Sangría: nos permite el desplazamiento de párrafos completos aumentando o disminuyendo la sangría.

Cita: nos permite una vez seleccionado un párrafo, mostrarlos como una cita (hará, dependiendo del estilo, un efecto de sangrado a la derecha e izquierda).

    Formato: Nos permite utilizar diferentes estilos en el proyecto. Si no se especifica el tamaño de la fuente, vendrá determinado por el estilo que se haya seleccionado.

Tamaño de la fuente. Si no se especifica el tamaño de la fuente, vendrá determinado por el estilo que se haya seleccionado.

Tipo de letra. Si no especificamos ninguno, se utilizará el tipo de letra del estilo seleccionado.

Color del texto o color de fondo. Pulsando sobre las pequeñas flechas a la derecha de los iconos, podemos seleccionar el color del texto o el color del fondo.

 Subíndice y superíndice: nos permite insertar subíndices y superíndices.

Pantalla completa: nos permite cambiar el modo a pantalla completa

Segunda fila

En esta fila nos encontraremos los elementos fundamentales para pegar texto desde documentos, introducir imágenes y elementos multimedia como vídeos, recursos tipo prezy, slideshare ... y todo aquello que permita ser compartido en la web. 

Para saber más, muestra retroalimentación:

Retroalimentación

Su funcionamiento es similar al de cualquier procesador de textos. Permite copiar, pegar, insertar multimedia... seleccionando la opción adecuada.

Opciones:

Nos permite deshacer (Ctrl + Z) o rehacer (Ctrl + Y) el último cambio realizado.

Nos permite cortar (Ctrl + X) el texto seleccionado (que desaparece) para posteriormente pegarlo.

Nos permite copiar (Ctrl + C) el texto seleccionado en el portapapeles.

Nos permite pegar (Ctrl + V) del portapapeles el texto previamente copiado o cortado. Debemos tener en cuenta que en esta modalidad de pegado, copiaremos el código "basura" del documento original, lo que podrá darnos problemas posteriormente a la hora de modificar el formato o texto pegado. Para limpiar este código, seleccionaremos el texto y utilizaremos los botones de "limpiar formato" o "limpiar código basura".

IMPORTANTE: No se pegarán las imágenes copiadas utilizando la opción de pegado (Ctrl + V)

Pegar como texto plano. Esta modalidad de pegado es útil cuando copiamos (o cortamos) desde una página web y queremos pegar el texto pero NO los formatos y estilos de origen.

Pegar contenido desde aplicaciones externas, tipo LibreOffice, OpenOffice...

     Permite embeber código HTML. 

Si no se desea ver todo el código HTML, se pueden embeber elementos con el botón  "Pegar fragmento de código HTML (código embebido)". Los pasos serán los siguientes:

    • Localizar, en páginas como Goear, Prezi, Slideshare, etc el objeto que se desea incluir.
    • Localizar la etiqueta "Insertar", "Compartir" o "Embed" que se encuentra junto al elemento.
    • Seleccionar y copiar el texto contenido (Ctrl+C). 
    • Dentro de eXe, crear un iDevice y localizar el icono "Pegar fragmento de código HTML (código embebido)".
    • Aparecerá una ventana donde deberemos copiar el código (Ctrl + V).
    • Pulsar el botón "Insertar".

Es habitual utilizarlo si has copiado el texto desde una fuente externa como el procesador de textos Word o Impress ya que en muchas ocasiones esta copia de texto puede añadir código innecesario que es posible suprimir sin problemas.

Windows Media: formatos wmv, wma. Será necesario tener instalados en el navegador los plugins correspondientes.

iFrame: ideal para introducir vídeos de YouTube o Vimeo aportando el enlace.

El editor de eXe Learning facilita la utilización de tablas mediante los botones de su barra de herramientas.

Opciones:

 Insertar una tabla nueva.

Cuando tengamos seleccionada una tabla, se activarán los siguientes iconos del editor:

Inserta una fila en la parte inferior.

Cuando se crea una nueva tabla aparecerá una ventana que permitirá su configuración:
 
 
Columnas: número de columnas de la tabla.

Filas: número de filas.

Cellpadding: espacio entre los contenidos de una celda y los bordes de la misma. Debe ser un número entero.
  
Cellspacing: espacio entre celdas contiguas. Debe ser un número entero.

Alinear: alineación de la tabla con respecto a la página (izquierda, centrada o a la derecha).

Borde: grosor del los bordes de las celdas. Si está a cero, no aparecerá borde.

Ancho: anchura de la tabla. Podemos indicar un número entero, o bien un porcentaje. En este último caso, se indica el porcentaje que ocupará la tabla con respecto al ancho de la página.

Alto: altura de la tabla.
